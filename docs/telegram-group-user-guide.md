# คู่มือผู้ใช้: เพิ่ม Telegram Bot เข้ากลุ่มเพื่อรับแจ้งเตือนออเดอร์

คู่มือนี้สำหรับผู้ใช้/แอดมินที่ต้องการให้ระบบส่ง "แจ้งเตือนออเดอร์" ไปยังกลุ่ม Telegram

## สิ่งที่ต้องมี
- Telegram Bot Token (ได้จาก BotFather)
- กลุ่ม Telegram ที่ต้องการรับแจ้งเตือน
- เข้าใช้งานหน้า `ตั้งค่า > แจ้งเตือนออเดอร์` ได้

---

## ขั้นตอนที่ 1: สร้าง Bot Token (ถ้ายังไม่มี)
1. เปิด Telegram แล้วค้นหา `@BotFather`
2. พิมพ์ `/newbot`
3. ตั้งชื่อบอทและ username ของบอท
4. BotFather จะส่ง `Token` มาให้ (รูปแบบ `123456789:AA...`)

เก็บ Token นี้ไว้ใช้ตอนเพิ่ม Telegram Sender Bot ในระบบ

---

## ขั้นตอนที่ 2: เพิ่ม Telegram Sender Bot ในระบบ
1. เข้าเมนู `ตั้งค่า > แจ้งเตือนออเดอร์`
2. ส่วน `Telegram Sender Bots` กด `เพิ่ม Telegram Bot`
3. กรอก:
   - ชื่อบอท
   - Bot Token
4. กด `บันทึก`

ผลลัพธ์ที่ควรได้:
- บันทึกสำเร็จ
- ระบบตั้งค่า webhook ให้อัตโนมัติ

ถ้าบันทึกไม่ผ่าน:
- ตรวจ Token ว่าถูกต้อง
- ตรวจว่าโดเมนระบบเป็น HTTPS และเข้าจากภายนอกได้

---

## ขั้นตอนที่ 3: เพิ่มบอทเข้ากลุ่ม Telegram
1. เปิดกลุ่ม Telegram ปลายทาง
2. กด `Add members` แล้วเลือกบอทที่สร้างไว้
3. แนะนำให้ตั้งบอทเป็นแอดมิน (อย่างน้อยสิทธิ์ส่งข้อความ)
4. ส่งข้อความ 1 ข้อความในกลุ่ม

จากนั้นกลับหน้า `แจ้งเตือนออเดอร์` แล้วกด `รีเฟรช`

---

## ขั้นตอนที่ 4: สร้างช่องทางแจ้งเตือนแบบ Telegram Group
1. กด `สร้างช่องทาง`
2. เลือกปลายทาง `Telegram Group`
3. เลือก:
   - `Telegram Bot (ผู้ส่ง)`
   - `Telegram Group (ปลายทาง)`
4. ตั้งค่าตามต้องการ:
   - รับออเดอร์จากบอทใด
   - Realtime หรือ Scheduled
   - รูปแบบข้อความ
5. กด `บันทึก`
6. กดปุ่ม `ทดสอบส่ง` ที่รายการช่องทาง

ถ้าทดสอบสำเร็จ แปลว่าใช้งานจริงได้

---

## ขั้นตอนที่ 5: ตรวจสอบการทำงาน
- เมื่อมีออเดอร์ใหม่: ควรมีข้อความแจ้งเตือนเข้ากลุ่ม
- ถ้าออเดอร์มีรูป: ระบบจะส่งรูปแนบไปด้วย
- ถ้าตั้ง Scheduled: ควรได้ข้อความสรุปตามเวลาที่กำหนด

---

## แก้ปัญหาเบื้องต้น

### 1) ไม่เห็นกลุ่มใน dropdown
- บอทยังไม่ได้อยู่ในกลุ่ม
- ยังไม่มี event เข้าระบบ (ลองส่งข้อความอีกครั้ง)
- ยังไม่กดรีเฟรช

### 2) กดทดสอบส่งแล้วไม่เข้า
- ช่องทางหรือบอทถูกปิดใช้งาน
- บอทไม่มีสิทธิ์ส่งในกลุ่ม
- เลือกกลุ่มผิด

### 3) รูปไม่แนบ
- ลิงก์รูปจากระบบต้องเข้าถึงได้จากภายนอก
- โดเมนระบบควรเป็น HTTPS
